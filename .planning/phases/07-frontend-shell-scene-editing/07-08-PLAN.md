<objective>
Deliver the acoustic-authoring surface (SC3): the isolation-spectrum editor (curve + table, server-owned interpolation, promote-to-twelfth), semi-transparent screen assignment (WEB-08/SCN-01), per-façade building spectra keyed by stable per-edge UUIDs with the load-bearing ring-diff recovery (D-02/WEB-09/SCN-02), and source SPL-at-reference calibration derived server-side (WEB-02). Purpose: an index-keyed façade spectrum silently re-points at the wrong façade after any vertex insert — a data-corruption bug no test would catch. UUID keying + the ring-diff make it structurally impossible; this is the phase's highest-risk logic and gets dedicated unit tests.
Output: `store/edges.ts` (ring diff + tests), `spectrum/SpectrumEditor.tsx`, `spectrum/interpolateClient.ts`, `panels/FacadePanel.tsx`, SPL calibration in `SourceFields.tsx`, a spectrum E2E.
