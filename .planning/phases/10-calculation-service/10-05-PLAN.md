## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| user input -> CalcPanel | Spacing + Run/Abort intents cross here; the guardrail + capability gate mediate before a job starts |
| CalcPanel -> DOM | Dynamic strings (reason, guardrail detail, counts) render as React text children |
| test harness -> network | Playwright intercepts all network so the UAT is offline (zero egress) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-10-05-01 | Tampering | dynamic strings to the DOM | mitigate | All reason/guardrail/count strings are React text children; grep gate asserts no innerHTML/dangerouslySetInnerHTML (the sibling-panel discipline) |
| T-10-05-02 | Denial of service | runaway grid submitted from UI | mitigate | Run is disabled while the guardrail is blocked and while not cross-origin isolated; the disabled-reason is always shown (no silent block) |
| T-10-05-03 | Repudiation | false green / silent stall | mitigate | The capability-failure banner is a distinct honest state (not a generic failed); cancelled keeps completed tiers; failed renders the real reason |
| T-10-05-04 | Information disclosure | test network egress | mitigate | Playwright routes ALL network; the UAT proves a network-off run, matching the Phase-8 zero-egress lineage |
</threat_model>
