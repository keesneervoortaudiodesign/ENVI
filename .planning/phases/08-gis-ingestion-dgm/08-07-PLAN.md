| network → OPFS | fetched GIS bytes are persisted client-side; only ingestion touches the network |
| imported features → scene | imported GeoJSON passes the same server-side validation as drawn features |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-08-07-02 | Tampering (path traversal in cache) | opfs.ts | mitigate | Project-UUID-keyed fixed layout; no user-controlled path segments |
